# waf-hth-lab

这是一个用于归档和发布多个独立 HTML 页面的仓库。页面内容来源于不同场景下的静态网页资源，作为一份技术性的存档与展示集合。

## 项目简介

本仓库不针对任何单个域名或具体网站，仅用于存放和展示若干独立的 HTML 文件。这些页面可能用于演示、测试、记录或学习目的，结构简单，不依赖外部后端服务。

## 目录说明

```
waf-hth-lab/
├── index.html          # 仓库首页（可选）
├── pages/              # 独立 HTML 页面归档目录
│   ├── example1.html
│   ├── example2.html
│   └── ...
└── README.md           # 本文件
```

- `pages/` 目录下存放所有归档的 HTML 页面，每个页面均为独立文件，可直接通过浏览器打开。
- 根目录下的 `index.html` 可作为导航页或入口文件（如有需要）。

## 页面归档说明

- 所有页面均为静态 HTML，不含服务器端脚本。
- 页面可能包含 CSS、JavaScript 等前端资源，但尽量内联或使用相对路径引用。
- 归档内容可能来自不同时间或场景，不保证彼此之间的连贯性。
- 每个页面在其文件头部或注释中会标注简要说明（如有）。

## 维护说明

- 本仓库由维护者不定期更新，新增或移除页面均基于实际需求。
- 欢迎通过 Issues 提交建议或反馈，但请注意本仓库不处理与具体网站相关的问题。
